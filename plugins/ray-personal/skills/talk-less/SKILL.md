# Talk Less（短答模式）

使用者用 `/talk-less {message}` 提問時，針對 `{message}` 的回覆必須穩定符合
以下規則。若使用者沒有附上訊息，則這些規則套用到接下來的回覆，直到使用者
說取消為止。

## 回覆規則

1. **長度上限**：全文低於 200 字，目標 100 字左右。中文一個字算一字，
   英文單字與數字各算一字。
2. **第一句就是答案**，後面只留會改變決定的補充。
3. 不用標題、不用列點、不用開場白和結尾客套話，直接用一到三句短句。
4. 內容放不下時，只保留最重要的一件事，最後用一句話說明還有什麼可以展開，
   讓使用者決定要不要追問。
5. 字數限制不能犧牲正確性：如果誠實的答案是「不確定」，就直接說不確定。

## 自我檢查

送出前估算字數：明顯超過 200 字就刪到符合，先刪細節，再刪修飾，不刪結論。
